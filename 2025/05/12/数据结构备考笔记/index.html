1 Notes

记录一下在考卷中遇到的自己不熟的内容：

    int end;//终点
    edgeNode *next;
    edgeNode(int e, edgeNode *n= NULL) {

    }
}

        head = h;
    }
}

class adjListGraph{
private:
    verNode* verList; //图中结点组成的动态数组
